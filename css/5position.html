  • position:
    1. static=>defualt(offset(top,right,bottom,left),z-index properties have no effect)==>z-index(specifies the stack level)
    2. relative(offset relative to its own boundary)==>offset does not effect position of other elements
    3. absolute(offset relative to its closest positined(non-static) ancestors otherwise viewport) ==>removed from normal document flow and no space is created in document for element
    4. fixed(offset relative to the viewport)==>removed from normal document flow and no space is created in document for element//NOTE:element placed at same position on every page
    5. sticky(offset relative to its nearest scrolling block ancestor(eg:viewport))==>ramain in normal document flow //offset does not effect position of other elements//NOTE:after a thershold point treated as fixed until it reaches the boundary of its parent=>except this treated as relative
  • z-index:
      Overlapping elements with a larger z-index cover those with a smaller one(overlapping elements condition)
